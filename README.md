# project-drone

STM32 기반 쿼드콥터 개발 프로젝트입니다. 기체 프레임 설계부터 전장 배치, 펌웨어 설정, 센서 캘리브레이션, 라즈베리파이 기반 확장까지 단계적으로 구현합니다.

## 목표

- 안정적인 호버링 및 자세 유지
- 영상 전송 기능 구현
- GPS 기반 자율비행 확장
- 장애물 감지 및 회피 기능 실험

## 하드웨어 구성

| 구분 | 부품 |
| --- | --- |
| FC | STM32H7 Nucleo |
| 모터 | RS2205 2300KV x4 |
| ESC | BLHeli_S 30A x4 |
| IMU | MPU-6500 |
| 기압계 | BMP388 |

## 개발 로드맵

### STEP 1. 프레임 설계

- [ ] 드론 크기 결정: RS2205 + 5045 프롭 기준, 225mm 전후
- [ ] Arm 설계: 16 x 19 mm M3 마운팅 홀, 최소 4mm 두께, PETG 소재
- [ ] 메인 플레이트 설계: FC 마운팅(M3 30.5 x 30.5mm 표준), ESC 배치 공간 확보
- [ ] 라즈베리파이 마운트 공간 확보
- [ ] 카메라 마운트 슬롯 설계
- [ ] 고무 댐퍼용 FC 마운트 보스 설계
- [ ] CAD 소프트웨어 선정: Fusion 360 또는 FreeCAD

### STEP 2. 전장 배치

- [ ] FC 선정: MPU-6500 IMU 직접 연결 가능, Betaflight 호환 F4/F7 계열
- [ ] BMP388 배치: 기류 간섭이 적은 위치, 스펀지 커버 적용
- [ ] 개별 30A ESC 배치 공간 설계
- [ ] 배터리 위치 설계: 무게중심(CG) 기준 중앙 배치, 스트랩 슬롯 반영
- [ ] 전원 분배 방식 결정: PDB 또는 배터리 패드 직결
- [ ] 라즈베리파이 전원 설계: 5V BEC 또는 별도 UBEC

### STEP 3. 조립 및 배선

- [ ] 모터 장착: M3 x 6mm 볼트, 열박음 인서트 사용
- [ ] FC 장착: 고무 댐퍼 4개로 진동 절연
- [ ] ESC와 모터 3상 배선
- [ ] 배터리 메인 파워 배선: 14AWG 이상, XT60 커넥터
- [ ] FC와 ESC DSHOT 신호선 배선
- [ ] 수신기(RX) 연결: SBUS 또는 CRSF, FC UART 포트 지정
- [ ] 라즈베리파이 통신용 UART 포트 예약
- [ ] 배터리 연결 전 멀티미터로 쇼트 체크

### STEP 4. 펌웨어 설정

- [ ] Betaflight Configurator 설치 및 FC 플래싱
- [ ] 보드 타겟 및 UART 포트 역할 설정
- [ ] 모터 방향 및 순서 확인
- [ ] DSHOT 프로토콜 설정: DSHOT600 권장
- [ ] 수신기 프로토콜 설정: SBUS 또는 CRSF
- [ ] MPU-6500 자이로 감지 확인
- [ ] BMP388 기압계 활성화
- [ ] Failsafe 동작 설정
- [ ] MSP over UART 활성화
- [ ] 펌웨어 방향 결정: Betaflight, ArduPilot, iNav 비교

### STEP 5. 센서 캘리브레이션

- [ ] 가속도계 캘리브레이션
- [ ] 자이로 캘리브레이션
- [ ] 자기장계 캘리브레이션
- [ ] BMP388 기압계 캘리브레이션
- [ ] ESC 스로틀 범위 캘리브레이션
- [ ] RPM 필터 설정
- [ ] Blackbox 로그 분석 및 필터 튜닝

### STEP 6. 라즈베리파이 확장

- [ ] 라즈베리파이와 FC 연결: MSP 또는 MAVLink over UART
- [ ] 카메라 모듈 구성: Pi Camera 또는 USB 카메라
- [ ] GStreamer 또는 FFmpeg 기반 영상 스트리밍
- [ ] WiFi/LTE 기반 원격 제어 실험
- [ ] OpenCV 기반 객체 추적 및 장애물 감지
- [ ] DroneKit-Python 또는 MAVSDK 기반 자율비행 실험
- [ ] 비행 로그 시각화 및 텔레메트리 대시보드 구성
