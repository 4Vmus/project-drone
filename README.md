# project-drone
# STM32 기반 쿼드콥터 개발 프로젝트

## 목표
- 호버링 자세유지
- 영상전송
- GPS 자율비행
- 장애물 회피

## 하드웨어
- FC: STM32H7 Nucleo
- 모터: RS2205 2300KV x4
- ESC: BLHeli_S 30A x4
- IMU: MPU-6500
- 기압계: BMP388

## 개발 단계
- [ ] STEP1: DSHOT300 모터 제어
- [ ] STEP2: IMU 드라이버
- [ ] STEP3: PID 자세 제어
- [ ] STEP4: 실제 비행
